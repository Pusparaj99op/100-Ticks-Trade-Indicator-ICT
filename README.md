# 100 Ticks Trade Indicator ICT

A .NET 8 application implementing ICT (Inner Circle Trading) methodology with 100-tick trade indicators for technical analysis.

## Project Structure

- `_100_Ticks_Trade_Indicator_ICT.cs` - Main indicator implementation
- `100 Ticks Trade Indicator ICT.csproj` - Project configuration

## Requirements

- .NET 8 or higher
- Visual Studio 2026 or compatible IDE

## Getting Started

1. Clone this repository
2. Open the solution file in Visual Studio
3. Build the project
4. Run the application
